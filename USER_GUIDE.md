## Getting Started

### System Requirements
- Modern web browser (Chrome, Firefox, Safari, Edge)
- Internet connection
- JavaScript enabled

### Accessing the Platform
1. Open your web browser
2. Navigate to the platform URL
3. Create an account or log in with existing credentials

### Account Types
The platform supports three types of users:

- **Customer**: Place and track delivery orders
- **Courier**: Accept and fulfill delivery orders
- **Administrator**: Manage the platform and users

## Customer Guide

### Creating an Account
1. Click "Sign Up" on the homepage
2. Fill in your details:
   - Email address
   - Password (minimum 8 characters)
   - Full name
   - Phone number
3. Click "Create Account"
4. Verify your email address (if required)

### Logging In
1. Click "Log In" on the homepage
2. Enter your email and password
3. Click "Sign In"

### Placing an Order

#### Step 1: Create New Order
1. From your dashboard, click "Create New Order"
2. Fill in the order details:
   - **Pickup Address**: Where the item should be collected
   - **Delivery Address**: Where the item should be delivered
   - **Distance**: Use the slider to set the distance in kilometers
3. Review the calculated price
4. Click "Place Order"

#### Step 2: Order Confirmation
- You'll receive an order confirmation with:
  - Order number
  - Pickup and delivery addresses
  - Total price
  - Estimated delivery time

### Tracking Your Order

#### Order Status Meanings
- **Created**: Order placed, waiting for courier assignment
- **Assigned**: Courier has been assigned to your order
- **Picked Up**: Courier has collected your item
- **In Transit**: Courier is on the way to delivery location
- **Delivered**: Order completed successfully

#### Real-Time Tracking
1. Go to "My Orders" from your dashboard
2. Click on an active order
3. View real-time status updates
4. See progress bar showing completion percentage

### Managing Your Orders

#### View Order History
1. Navigate to "Order History"
2. Filter orders by:
   - Status
   - Date range
   - Price range

#### Cancel an Order
1. Go to "My Orders"
2. Find the order you want to cancel
3. Click "Cancel Order" (only available for orders not yet picked up)
4. Confirm cancellation

### Notifications
- Receive notifications for:
  - Order status changes
  - Courier assignment
  - Delivery completion
- Check notifications in the bell icon menu
- Mark notifications as read

### Account Settings
1. Click on your profile picture/name
2. Select "Account Settings"
3. Update:
   - Personal information
   - Contact details
   - Password
   - Notification preferences

## Courier Guide

### Getting Started as a Courier
1. Contact an administrator to create your courier account
2. Log in with provided credentials
3. Complete your profile setup

### Dashboard Overview
Your courier dashboard shows:
- Available orders
- Your current orders
- Earnings summary
- Performance metrics

### Managing Availability

#### Setting Your Status
1. Use the availability toggle on your dashboard
2. **Available**: You can receive new order assignments
3. **Unavailable**: You won't receive new orders

#### Location Updates
1. Update your current location description
2. This helps with efficient order assignment

### Accepting Orders

#### Viewing Available Orders
1. Go to "Available Orders"
2. See list of unassigned orders with:
   - Pickup and delivery addresses
   - Distance
   - Payment amount
   - Estimated time

#### Accepting an Order
1. Click "Accept" on an available order
2. Confirm acceptance
3. Order moves to "My Orders"

### Fulfilling Orders

#### Order Workflow
Follow these steps for each order:

1. **Accept Order**
   - Review order details
   - Plan your route

2. **Go to Pickup Location**
   - Navigate to pickup address
   - Contact customer if needed

3. **Update Status to "Picked Up"**
   - Click "Mark as Picked Up" when you collect the item
   - This notifies the customer

4. **Travel to Delivery Location**
   - Navigate to delivery address
   - Update status to "In Transit"

5. **Complete Delivery**
   - Deliver the item
   - Click "Mark as Delivered"
   - Order is complete

#### Status Updates
Always update order status promptly:
- Keeps customers informed
- Maintains platform reliability
- Affects your performance rating

### Managing Your Orders

#### Current Orders
- View all assigned orders
- See pickup and delivery details
- Access customer contact information
- Update order status

#### Order History
- Review completed orders
- Check earnings per order
- View performance metrics

### Earnings and Performance

#### Earnings Tracking
- View daily, weekly, and monthly earnings
- See payment breakdown per order
- Track payment status

#### Performance Metrics
- Delivery time averages
- Customer ratings
- Completion rate
- Reliability score

## Troubleshooting

### Common Issues

#### Login Problems
**Problem**: Can't log in to account
**Solutions**:
1. Check email and password spelling
2. Use "Forgot Password" to reset
3. Clear browser cache and cookies
4. Try a different browser
5. Contact support if issue persists

#### Order Issues
**Problem**: Order not updating
**Solutions**:
1. Refresh the page
2. Check internet connection
3. Log out and log back in
4. Contact courier directly (customers)
5. Contact support

#### Payment Issues
**Problem**: Pricing seems incorrect
**Solutions**:
1. Check distance calculation
2. Review current pricing rates
3. Contact administrator
4. Check for recent pricing updates

#### Notification Problems
**Problem**: Not receiving notifications
**Solutions**:
1. Check notification settings
2. Ensure browser allows notifications
3. Check email spam folder
4. Update notification preferences
5. Contact support

### Browser Compatibility
- **Recommended**: Chrome 90+, Firefox 88+, Safari 14+, Edge 90+
- **Minimum**: Chrome 80+, Firefox 75+, Safari 13+, Edge 80+

### Performance Issues
If the platform is running slowly:
1. Check your internet connection
2. Close unnecessary browser tabs
3. Clear browser cache
4. Restart your browser
5. Try a different browser

### Mobile Usage
- Platform is optimized for mobile browsers
- Use landscape mode for better experience
- Ensure stable internet connection
- Update your mobile browser

## Frequently Asked Questions

### General Questions

**Q: How do I change my password?**
A: Go to Account Settings > Security > Change Password. Enter your current password and new password twice.

**Q: Can I use the platform on my mobile phone?**
A: Yes, the platform is fully responsive and works on mobile browsers.

**Q: How do I contact support?**
A: Use the "Contact Support" link in the help menu or email support@deliveryplatform.com.

### Customer Questions

**Q: How is the delivery price calculated?**
A: Price = Base fee + (Distance in km Ã— Per-km rate). Current rates are displayed when creating an order.

**Q: Can I cancel an order after placing it?**
A: Yes, you can cancel orders that haven't been picked up yet. Go to "My Orders" and click "Cancel Order."

**Q: How long does delivery usually take?**
A: Delivery times vary based on distance and courier availability. Estimated times are shown when placing orders.

**Q: What if my order is delayed?**
A: Contact the assigned courier directly or reach out to customer support for assistance.

**Q: Can I track my order in real-time?**
A: Yes, order status updates in real-time. You'll see progress from "Created" to "Delivered."

### Courier Questions

**Q: How do I become a courier?**
A: Contact an administrator to apply for a courier account. You'll need to provide identification and contact information.

**Q: When do I get paid?**
A: Payment schedules are set by administrators. Check your earnings dashboard for payment status.

**Q: Can I reject an order after accepting it?**
A: Contact an administrator immediately if you need to cancel an accepted order. This should be avoided when possible.

**Q: What if I can't find the pickup or delivery location?**
A: Contact the customer directly using the provided contact information, or reach out to support.

**Q: How do I report an issue with an order?**
A: Use the "Report Issue" button on the order details page or contact support directly.

### Administrator Questions

**Q: How do I add new couriers to the system?**
A: Go to User Management > Create New User, select "Courier" role, and fill in the required information.

**Q: Can I change pricing after orders are placed?**
A: Pricing changes only affect new orders. Existing orders maintain their original pricing.

**Q: How do I generate reports?**
A: Go to Analytics section and select the type of report you need. You can filter by date range and export data.

**Q: What if a courier is not performing well?**
A: Review their performance metrics, contact them directly, provide additional training, or deactivate their account if necessary.

**Q: How do I handle customer complaints?**
A: Review the order details, contact involved parties, investigate the issue, and take appropriate action (refund, reassignment, etc.).

### Technical Questions

**Q: What browsers are supported?**
A: Modern versions of Chrome, Firefox, Safari, and Edge. See the troubleshooting section for specific version requirements.

**Q: Is my data secure?**
A: Yes, we use industry-standard security measures including encryption, secure authentication, and regular security audits.

**Q: Can I integrate with other systems?**
A: API documentation is available for integration purposes. Contact administrators for access and documentation.

**Q: What if the website is down?**
A: Check our status page or contact support. We monitor system availability 24/7.
